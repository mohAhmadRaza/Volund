
# Volund - AI Chatbot

## Introduction
Welcome to Volund, an AI chatbot designed to help users achieve their goals, resolve challenges, and provide detailed solutions. Volund leverages the power of OpenAI's GPT-4 to deliver intelligent and context-aware responses. This project was developed during the International Hackathon "Build Your AI Startup."

## Features
- **Goal-Oriented Responses:** Volund assists users by understanding their goals and providing tailored solutions.
- **Challenge Resolution:** It helps users overcome challenges by offering practical advice and resources.
- **Resource Integration:** Provides links to relevant YouTube videos and other resources for comprehensive assistance.
- **User-Friendly Interface:** Built using Streamlit for an intuitive and interactive user experience.

## Technologies Used
- **Streamlit:** For the frontend interface.
- **CSS & HTML:** For styling and layout.
- **Python:** The core programming language for backend logic.
- **OpenAI GPT-4:** For generating intelligent and context-aware responses.
- **Gemini API:** For input processing and integration.

## Getting Started

### Prerequisites
- Python 3.7 or later
- Streamlit
- OpenAI Python package
- Requests library

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mohAhmadRaza/Volund.git
   cd Volund
   ```

2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

3. Set up your OpenAI API key:
   ```bash
   export OPENAI_API_KEY='your-api-key-here'
   ```

### Running the App

To start the Streamlit app, navigate to the project directory and run:
```bash
streamlit run app.py
```

The app will be accessible at `volund.streamlit.app`.

## Usage
1. Open the app in your web browser.
2. Enter your goals or challenges in the input field.
3. Volund will process your input and provide a detailed response along with a link to a relevant YouTube video.
